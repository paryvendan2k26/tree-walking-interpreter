# Mini Language Interpreter in C++

A simple programming language interpreter built in C++ using:

- Lexer
- Recursive Descent Parser
- AST (Abstract Syntax Tree)
- Tree-Walk Interpreter
- REPL (Read-Eval-Print Loop)

This project is inspired by the interpreter architecture explained in the AOSA article:

> Building an Interpreter / A Python Interpreter Written in Python

---

# Features

Currently the language supports:

- Integer arithmetic
- Variables
- Assignments
- Operator precedence
- Comparisons (`>`, `<`, `==`)
- `if / else`
- `while` loops
- Functions
- Function calls with parameters
- Local function scope
- Interactive REPL
